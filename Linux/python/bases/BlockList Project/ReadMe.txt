в рабочей папке проекта находятся вложеные каталоги, которые отвечают за определенные источники (внутри каждого католога-источника есть скрипт, который парсит свой источник)  

так же в рабочей папке проекта находятся 3 основных скрипта, которые запускаются в следующей последовательности:
1. StartParse.py - запускает по очереди парсинг всех источников (внутри источников создаются файлы csv) 
2. StartUpdate.py - находит изменения в источниках по результатам парсинга (внутри источников создает Update.csv и log.txt)
3. RestAPI.py - отсылает изменения источников в базу.
